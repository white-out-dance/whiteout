    Host Flow

    Quick Start
    1. Open Host, sign in, and create a room.
    2. Open Whiteout on the booth machine, enter the room code and DJ key, then connect.
    3. Throw the QR on screen and let guests hit the mobile request wall.
    4. Guests can vote requests up or down while the DJ approves, plays, or rejects them live.

    Tip: the booth QR posters already open the guest page with the room code filled in.
